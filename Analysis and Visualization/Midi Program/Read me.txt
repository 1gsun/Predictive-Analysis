Run main.py to:1. Load a database (Classified Midis). During the process, genre, artist name, and midi name will be written in the dataframe as well2. Prompt user for a choice: 1) Use pre-loaded database 2) Import a new midi file by giving the file path 3) Exit the program 3. Plot 4 charts to visualize the basic information The midi file is re-processed to a dataframe. During the process, some messages were thrown away, filtered, or interpreted for the efficiency. Please see details in the midi_to_dataframe.py.
